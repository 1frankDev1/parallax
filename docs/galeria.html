Gallery Integration

To display your gallery on any website, follow these two simple steps:

1. Include the script

Add this script tag inside your <head> or at the end of your <body>:

<script src="https://menutech.xyz/assets/js/menutechUI.js"></script>

2. Add the custom element

Place this tag exactly where you want the gallery to appear. Replace yourdomain.com with the domain registered in the Admin Panel.

<menutech-gallery domain="yourdomain.com"></menutech-gallery>
