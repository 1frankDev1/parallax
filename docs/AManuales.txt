SOLUCIÓN A ERRORES DE ESQUEMA (Missing Column 'category'):
---------------------------------------------------------
Si al intentar guardar recibes un error que dice "column 'category' does not exist", es porque el caché de Supabase aún no reconoce los cambios.

Para solucionarlo, copia y ejecuta TODO este bloque de código en el "SQL Editor" de tu Dashboard de Supabase:

ALTER TABLE public.manual_content ADD COLUMN IF NOT EXISTS category TEXT DEFAULT 'General';
ALTER TABLE public.manual_content ADD COLUMN IF NOT EXISTS description TEXT;
NOTIFY pgrst, 'reload schema';

Esto forzará a Supabase a reconocer las nuevas columnas inmediatamente.
